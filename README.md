# ALUR

### **1. Inisialisasi Data Awal**  
   - Program langsung menginisialisasi 5 produk awal pada `main`.  
   - Setiap produk memiliki atribut:  
     - **ID**
     - **Nama Produk**
     - **Harga Produk** 
     - **Stok Produk**
     - **Foto Produk**
     - **Jenis, Bahan, Warna**
     - **Untuk, Size, Merk**

#### **Display**  
1️⃣ **Tambah Produk Baru**
